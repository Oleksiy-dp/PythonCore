Як працювати з git
    1. Клонувати репозиторій
git clone https://github.com/katemalash/Rv-087.PythonCore.git

    2. Створити вітку (від main) з такою назвою, як main_Ваше прізвище_ім’я
git checkout -b main_firstname_lastname

    3. Запушити створену вітку на GitHub
git push origin main_firstname_lastname

    4. Від своєї вітки створити вітку прізвище_ім’я_hw_2
git checkout -b firstname_lastname_hw2

    5. Створити папку для виконання усіх дз і назвати своїм прізвищем
mkdir lastname

    6. У цій папці створити папку HW2
cd lastname
mkdir HW2

    7. У неї додати файли з попереднім виконаним ДЗ

Далі алгоритм роботи власне з git

    8. Застежити усі файли 
git add <file_name>git 

git reset HEAD <file>…​ для исключения из индекса. 

    9. Зробити коміт
git commit -m “message”
    10. Запушити вітку з конкретним ДЗ
git push origin firstname_lastname_hw_2
